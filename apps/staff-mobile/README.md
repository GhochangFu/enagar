# @enagar/staff-mobile — STUB (Phase 4)

Field-staff mobile app for sanitation, sweeping, asset-survey, and grievance-verification crews:

- Geo-tagged photo capture, signed at the device (anti-tampering)
- Daily route + checklist
- Offline queue for low-connectivity wards
- SOS / panic button + supervisor chat

## Status

Phase-0 stub only. Implemented in Phase 4 alongside the grievance-redressal pipeline.
