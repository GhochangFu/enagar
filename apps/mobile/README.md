# @enagar/mobile — STUB (Phase 5/8)

Citizen-facing **React Native (Expo)** app. Targets:

- iOS 14+, Android 9+
- Offline-first for application drafts and grievance attachments (Phase 8)
- Native push (FCM/APNs), camera + GPS for grievances, BLE for parking gates (Phase 5)
- Shares logic with `@enagar/citizen-pwa` via `@enagar/sdk`, `@enagar/forms`, `@enagar/i18n`, `@enagar/types`; UI primitives via `@enagar/ui-native`

## Status

Phase-0 stub only — see ADR-0003 (parallel mobile + PWA, PWA-first lead) and `ROADMAP.md` Phase 5 / Phase 8.
