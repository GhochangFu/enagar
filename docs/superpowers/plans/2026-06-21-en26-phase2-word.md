# EN-26 Phase 2 — Word import Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add structured Word `.docx` table import (EN-37) and wire Excel + Word through the existing review wizard on both admin portals (EN-38).

**Architecture:** Mirror Phase 1 Excel: first table in the document with columns `field_id`, `label_en`, `type` (+ optional locale/options columns). Shared table-row parser; Word extractor reads `word/document.xml` via JSZip + XML parse. `FormImportService` dispatches by file type. `FormImportPanel` accepts `.xlsx` and `.docx`.

**Tech Stack:** NestJS API, `@enagar/forms/form-import`, JSZip, fast-xml-parser, `docx` (fixture generator only)
