## Exit Criteria

SSA-3 is complete when all are true:

1. **Registry:** Step 3 + scope `full`/`workflow` exposes only workflow tools (no form/config tools).
2. **Tools:** `applyWorkflowDraft`, `replaceWorkflowDraft`, `mergeWorkflowDraft`, `applyWorkflowTemplate` validate via `validateWorkflowDefinition` and enforce service code prefix before save.
3. **Templates:** `linear_approval` and `scrutiny` templates produce valid drafts for a tenant service code.
4. **Merge:** `mergeWorkflowDraft` upserts stages by `code` and merges transitions without dropping unrelated stages.
5. **SSE:** Tenant `POST .../message` works on step 3; emits `draft_updated` with layer `workflow`.
6. **Persistence:** Messages + audit logs written per turn on workflow step.
7. **UI:** Workflow-only scope — chat enabled on step 3, summary panel shows stage/transition counts, refreshes after tool success.
8. **Step completion:** Step 3 marked complete when `workflow_draft_valid` is green.
9. **Verification:** New unit/integration specs pass; existing SSA specs remain green.
10. **Review:** No open high-severity spec compliance issues.

## Manual UI Tests (operator)

**Prerequisites:** API + admin-tenant running, `SETUP_ASSISTANT_SKIP_DPA_DEV=true`, LLM configured, service with `workflow_pattern` `cert-issuance` or `linear`.

### Tenant workflow scope

1. Open Setup Assistant → **Workflow only** → confirm step 3 active, chat input enabled.
2. Ask: _"Apply the linear approval template"_ → `applyWorkflowTemplate: OK`, preview shows stages/transitions.
3. Ask: _"Replace with scrutiny template"_ → draft replaced, preview updates.
4. Ask: _"Add a clerk review stage after submitted"_ (merge) → stages increase, `mergeWorkflowDraft: OK`.
5. Readiness: `workflow_draft_valid` turns green; step 3 shows ✓.
6. Service Designer → workflow draft matches assistant changes.

### Full scope

7. Start **Full setup** session → navigate to step 3 → repeat one template apply; form step tools must not run on step 3.

### Security

8. Cross-tenant session/service → rejected.
