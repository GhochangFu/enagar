Typography options — eNagarSeba

Compare font pairings on the same Warm Coral (Tricolor Calm) tokens used in Citizen PWA (:3000), Tenant Admin (:3002), and State Admin (:3003). Bengali and Hindi always use dedicated Noto (or Plex Devanagari) — only Latin display/body pairs change. Buttons use the body stack at 500 weight with Lucide-style icon + label (not heading weight).

How to view — from repo root: npx serve docs/design-previews -p 8788 then open http://localhost:8788/typography-options-preview.html (or double-click the file). Compare at 100% zoom and on a phone width (~390px).

Button spec in this preview: 13px / weight 500 / 0.02em letter-spacing · 16px icon · 6px gap · primary filled + secondary outline (matches target for @enagar/ui Button refresh).

Implementation note: One shared Latin pairing should live in packages/config/styles/tricolor-calm.css (--tenant-font-family + new --platform-heading-font). Citizen PWA already splits heading vs body in globals.css; Tenant/State can adopt the same split in a UX sprint.
